7. The Popup Menu

The popup menu is activated whenever the right mouse button is clicked in the Battery icon area.

7.1 (Un)Dock

This docks or undocks Battery from the panel. This option will only work in KDE complient window managers like KWM. The default state is determined by the setup dialog.

7.2 Setup

This will activate the setup dialog. See the Setup Dialog section for more details.

7.3 Standby

Puts your laptop into standby state - this menu item only appears if the apmd package is installed in your system and either you are the root user or /usr/bin/apm is setuid root (see the section here on "Compilation and Installation" for information on how to install this package).

7.4 Suspend

Puts your laptop into suspend state - this menu item only appears if the apmd package is installed in your system and either you are the root user or /usr/bin/apm is setuid root (see the section here on "Compilation and Installation" for information on how to install this package).

7.5 Quit

This hides the battery monitor.
